 Expense Manager 💰

A responsive web-based expense tracking application that helps users manage income, expenses, budgets, and financial summaries efficiently.

 Live Demo
Click here to view the live project
https://yelugubanti-sunitha.github.io/expense-manager/

🚀 Features

✅ Add income and expense entries
✅ Real-time balance calculation
✅ Monthly expense summary
✅ Budget tracking with warning alerts
✅ Category-wise expense breakdown (Chart.js)
✅ Income vs Expense comparison chart
✅ Search functionality
✅ Filter by category
✅ Export data as CSV
✅ Dark Mode (saved using localStorage)
✅ Responsive design (Bootstrap 5)

🛠️ Technologies Used

- HTML5
- CSS3
- Bootstrap 5
- JavaScript (ES6)
- Chart.js
- LocalStorage API

📊 How It Works

- User entries are stored in the browser using localStorage.
- Data updates automatically without page refresh.
- Charts update dynamically based on entries.
- Each user’s data is stored locally and is not shared.

📌 Future Improvements

- Add authentication system
- Store data in a database
- Add recurring expenses
- Add yearly financial reports
